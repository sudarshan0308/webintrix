📟 Webnitrix: Galvanic OS 3.0.4
Webnitrix is a high-fidelity, interactive 3D recreation of the Omnitrix interface, built using Three.js and Galvanic technology. This project features a fully functional Master Control system, holographic alien displays, and immersive sound effects designed for both Desktop and Mobile environments.


🚀 [LIVE DEMO: Access the Omnitrix Here](https://sudarshan0308.github.io/webintrix/)

🛠️ System Features
Master Control Interface: Toggle between 10 unique alien DNA samples.

Holographic Projection: Dynamic 2D-to-3D sprite projection system with floating animation.

Responsive UI: Optimized for high-end laptops and mobile touch-screen devices.

Power Management: Visual and audio warnings when DNA stability drops below 30% (Red Alert Mode).

Galvanic Audio Engine: Immersive sound effects for transformation, selection, and system timeout.

🕹️ Controls
For Desktop
Left Click: Initialize System / Finalize Transformation.

Scroll Wheel: Rotate Dial to cycle through Alien DNA.

For Mobile
Tap Screen: Activate Core / Transform.

Vertical Swipe: Spin the dial to select your hero.

📝 Setup for Developers
If you want to run this locally:

Clone the repo: git clone https://github.com/sudarshan0308/webintrix.git

Open the folder in VS Code.

Use Live Server to run index.html.

Ensure the /static/ folder is present with all .glb, .png, and .mp3 assets.
